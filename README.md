# APK Extractor Pro

Утилита для извлечения всех ресурсов из APK файлов:
- 3D модели (Unity AssetBundle, OBJ, FBX)
- Текстуры (PNG, JPG, DDS)
- Аудио (MP3, WAV, OGG)
- Иконки приложений
- Unity ассеты и скрипты

## Установка

```bash
# Клонируйте или скачайте файлы
git clone https://github.com/yourusername/apk-extractor-pro.git
cd apk-extractor-pro

# Установите зависимости
chmod +x install.sh
./install.sh
```
